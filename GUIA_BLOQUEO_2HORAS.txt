📋 RESUMEN EJECUTIVO
────────────────────────────────────────────────────────────────────────────────
El sistema ahora bloquea automáticamente 2 horas consecutivas cuando se hace una
reserva. Si un cliente reserva las 10:00 AM, automáticamente se bloquea también
la franja de 12:00 PM, previniendo doble-booking.

🎯 OBJETIVO LOGRADO
────────────────────────────────────────────────────────────────────────────────
✅ Cada cita bloquea automáticamente un bloque de 2 horas
✅ El select de horas muestra las ocupadas como deshabilitadas
✅ Las horas ocupadas aparecen en rojo con etiqueta "(OCUPADA)"
✅ Se actualiza automáticamente al seleccionar una fecha
✅ La reserva se persiste en el objeto SISTEMA_DISPONIBILIDAD

═══════════════════════════════════════════════════════════════════════════════
⚙️ COMPONENTES DEL SISTEMA
═══════════════════════════════════════════════════════════════════════════════

1. ESTRUCTURA DE DATOS
────────────────────────────────────────────────────────────────────────────────
const SISTEMA_DISPONIBILIDAD = {
    reservas: [
        // Formato: { fecha: '2026-03-20', horaInicio: '10:00', proximaHora: '12:00', nombreCliente: '...', duracion: 2 }
    ],
    
    // Array que registra reservas con 2-hour blocking automático
}

2. MÉTODOS CLAVE
────────────────────────────────────────────────────────────────────────────────

✨ getProximaHora(hora)
   • Calcula la próxima hora (+2 horas)
   • Input:  '10:00' → Output: '12:00'
   • Input:  '14:00' → Output: '16:00'

✨ horaDisponible(fechaISO, hora)
   • Verifica si una hora está disponible
   • Comprueba horaInicio AND proximaHora en reservas[]
   • Si alguno está ocupado, la hora no está disponible

✨ agregarReserva(fechaISO, horaInicio, nombreCliente)
   • Agrega una reserva + bloquea 2 horas automáticamente
   • Validación: verifica horaDisponible() antes de guardar
   • Retorna: true si exitosa, false si conflicto

✨ getHorasOcupadas(fechaISO)
   • Retorna array de horas ocupadas para una fecha
   • Ejemplo: ['10:00', '12:00', '14:00', '16:00']
   • Incluye ambas horas del bloque reservado

3. INTERFAZ ACTUALIZADA
────────────────────────────────────────────────────────────────────────────────

actualizarHorariosDisponibles(fechaSeleccionada)
   • Lee las reservas del día seleccionado
   • Llama a getHorasOcupadas(fechaISO)
   • Desactiva (disabled) horas ocupadas en el select
   • Agrega "(OCUPADA)" a los textos de opciones
   • Aplica estilos visuales (rojo, tachado)

4. MANEJADOR DE FORMULARIO
────────────────────────────────────────────────────────────────────────────────

form.addEventListener('submit', ...)
   1. Valida todos los campos
   2. Extrae: fecha, hora, nombre, etc.
   3. Llama: SISTEMA_DISPONIBILIDAD.agregarReserva()
   4. Si exitosa: muestra confirmación + actualiza horarios
   5. Si falla: muestra alerta + cancela envío

⚡ CÓMO FUNCIONA EL FLUJO
═══════════════════════════════════════════════════════════════════════════════

1. USUARIO SELECCIONA FECHA
   └─→ onChange en Flatpickr se dispara
   └─→ Llama: actualizarHorariosDisponibles(fecha)

2. ACTUALIZAR HORARIOS DISPONIBLES
   └─→ Lee fecha seleccionada (ISO format)
   └─→ Obtiene horariosParaDia (7 para lunes-viernes, 6 para fin de semana)
   └─→ Lee getHorasOcupadas(fecha) desde reservas[]
   └─→ Para cada hora:
       • Si en horasOcupadas → disabled + "(OCUPADA)" + rojo
       • Si NO in horasOcupadas → enabled + verde
   └─→ Muestra select actualizado

3. USUARIO ENVÍA FORMULARIO
   └─→ Validar todos los campos
   └─→ Si válido:
       • Extrae fecha, hora, nombre
       • Llama: agregarReserva(fechaISO, horaInicio, nombre)
   
4. AGREGAR RESERVA
   └─→ Verifica: horaDisponible(fecha, hora)?
   └─→ Si NO disponible → return false, muestra alerta
   └─→ Si SÍ disponible:
       • Calcula: proximaHora = getProximaHora(horaInicio)
       • Agrega a reservas[]: { fecha, horaInicio, proximaHora, nombre, duracion: 2 }
       • Retorna: true
   
5. DESPUÉS DE RESERVA EXITOSA
   └─→ Muestra confirmación
   └─→ Actualiza horarios nuevamente → ahora muestra 2 horas bloqueadas
   └─→ Limpia formulario
   └─→ Select resetea y show "Selecciona una hora"
