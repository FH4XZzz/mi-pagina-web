🎓 CÓMO APRENDER A USAR EL SISTEMA
════════════════════════════════════════════════════════════════════════════

NIVEL 1 - USUARIO FINAL (5 minutos)
─────────────────────────────────
1. Abre index.html en tu navegador
2. Ve a "Reserva Tu Cita"
3. Haz clic en el campo de fecha
4. Observa:
   ✓ Calendario con días verdes (disponibles)
   ✓ Leyenda visual
   ✓ Información de horarios
5. Selecciona una fecha verde
6. Observa que los horarios se actualizan automáticamente

NIVEL 2 - ADMINISTRADOR (15 minutos)
───────────────────────────────────
1. Lee: RESUMEN_SISTEMA_COMPLETO.txt
2. Lee: EJEMPLOS_CODIGO.txt (hasta OPCIÓN 5)
3. Abre DevTools (F12)
4. Ejecuta uno de los ejemplos en la consola
5. Verifica que el calendario cambió
6. Éxito! Ya puedes gestionar disponibilidad

NIVEL 3 - DESARROLLADOR (30 minutos)
────────────────────────────────────
1. Lee: SISTEMA_DISPONIBILIDAD.txt (completo)
2. Lee: script.js (busca SISTEMA_DISPONIBILIDAD)
3. Lee: styles.css (sección de disponibilidad)
4. Experimenta con todos los EJEMPLOS_CODIGO.txt
5. Modifica horariosDisponibles en script.js
6. Entiende getHorariosParaDia() para modificar lógica
7. Opcional: Conecta con tu backend

NIVEL 4 - BACKEND (1-2 horas)
─────────────────────────────
1. Lee toda la documentación anterior
2. Crea un endpoint API para obtener disponibilidad
3. Modifica script.js para llamar a tu API
4. Sincroniza base de datos con SISTEMA_DISPONIBILIDAD
5. Implementa confirmación de reservas
6. Agrega notificaciones/emails
