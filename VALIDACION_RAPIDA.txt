═══════════════════════════════════════════════════════════════════════════════
🛟 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problema: Las horas no se desactivan
Solución:
  1. Abre Console (F12)
  2. Verifica: SISTEMA_DISPONIBILIDAD.reservas
  3. Debe mostrar array con al menos 1 elemento
  4. Si vacío, la reserva no se agregó
  5. Busca errores en Console

Problema: No veo "(OCUPADA)" en las opciones
Solución:
  1. Verifica que script.js esté actualizado (línea ~520)
  2. Busca: actualizarHorariosDisponibles
  3. Debe contener: option.textContent += ' (OCUPADA)'
  4. Si no está, recarga los archivos

Problema: Puedo seleccionar una hora ocupada
Solución:
  1. Verifica que index.html tiene id="hora" en select
  2. Verifica que script.js tiene disabled = true para horas ocupadas
  3. Comprueba DevTools → Elements, busca option que deba tener disabled
