📊 DESCRIPCIÓN DEL SISTEMA
════════════════════════════════════════════════════════════════════════════

Se ha implementado un sistema inteligente de disponibilidad similar a Airbnb/Booking
que permite gestionar dinámicamente:

✅ Días disponibles (indicador VERDE 🟢)
✅ Días ocupados (indicador ROJO 🔴)
✅ Horarios variables según día de semana
✅ Interfaz visual profesional
✅ Sin necesidad de backend

🎯 CARACTERÍSTICAS PRINCIPALES
════════════════════════════════════════════════════════════════════════════

1. CALENDARIO INTELIGENTE 📅
   ├─ Muestra días disponibles en VERDE
   ├─ Muestra días ocupados en ROJO
   ├─ Bloquea automáticamente días ocupados
   ├─ Iconos visuales (puntos de colores)
   └─ Tema oscuro profesional

2. HORARIOS DINÁMICOS ⏰
   ├─ Lunes-Viernes: 8:00 AM - 9:00 PM
   ├─ Sábado-Domingo: 10:00 AM - 9:00 PM
   ├─ Se actualizan al seleccionar una fecha
   └─ Compatible con PM/AM

3. LEYENDA VISUAL 🎨
   ├─ Disponible (Verde)
   ├─ Ocupado (Rojo)
   └─ Seleccionado (Dorado)

💻 CÓMO USAR EL SISTEMA
════════════════════════════════════════════════════════════════════════════

CONFIGURACIÓN BÁSICA (en script.js):

El objeto SISTEMA_DISPONIBILIDAD controla todo:

const SISTEMA_DISPONIBILIDAD = {
    diasOcupados: [
        // Agregar fechas ocupadas aquí
        // Formato: 'YYYY-MM-DD'
    ],
    
    horariosDisponibles: {
        normal: [...],      // Lunes-Viernes
        finDeSemana: [...]  // Sábado-Domingo
    }
}

MARCAR UN DÍA COMO OCUPADO:
═══════════════════════════════

Agregar en el array diasOcupados:

SISTEMA_DISPONIBILIDAD.diasOcupados = [
    '2026-03-20',  // 20 de marzo (se mostrará ROJO)
    '2026-03-21',  // 21 de marzo
    '2026-03-25'   // 25 de marzo
];

O usar la función:

// Marcar un día como ocupado
SISTEMA_DISPONIBILIDAD.agregarDiaOcupado('2026-03-20');

// Quitar un día de ocupado
SISTEMA_DISPONIBILIDAD.quitarDiaOcupado('2026-03-20');

CAMBIAR HORARIOS POR DÍA:
═════════════════════════

Modificar en script.js los arrays de horariosDisponibles:

normal: [
    { hora: '08:00', etiqueta: '8:00 AM', disponible: true },
    { hora: '09:00', etiqueta: '9:00 AM', disponible: true },
    ...
]

Puedes:
- Agregar más horas
- Quitar horas
- Marcar como disponible: true/false
- Cambiar etiqueta a lo que quieras

EJEMPLO AVANZADO - CONECTAR CON BACKEND:
════════════════════════════════════════

Si quieres obtener disponibilidad desde un servidor:

async function cargarDisponibilidad() {
    try {
        const respuesta = await fetch('/api/disponibilidad');
        const datos = await respuesta.json();
        SISTEMA_DISPONIBILIDAD.diasOcupados = datos.diasOcupados;
        SISTEMA_DISPONIBILIDAD.horariosDisponibles = datos.horarios;
    } catch (error) {
        console.error('Error al cargar disponibilidad:', error);
    }
}

🔧 MÉTODOS DISPONIBLES
════════════════════════════════════════════════════════════════════════════

getDiaOcupado(fechaISO)
├─ Parámetro: '2026-03-20' (string)
├─ Retorna: true/false
└─ Uso: Verificar si un día está ocupado

getHorariosParaDia(fecha)
├─ Parámetro: Date object
├─ Retorna: Array de horarios
└─ Uso: Obtener horarios según el día (considera fin de semana)

agregarDiaOcupado(fechaISO)
├─ Parámetro: '2026-03-20' (string)
├─ Retorna: void
└─ Uso: Marcar un día como ocupado

quitarDiaOcupado(fechaISO)
├─ Parámetro: '2026-03-20' (string)
├─ Retorna: void
└─ Uso: Desmarcar un día como ocupado

   ✓ Debe estar deshabilitada
   ✓ Debe aparecer en rojo/tachada
   ✓ No debe poder seleccionarla
