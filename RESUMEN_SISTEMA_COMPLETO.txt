🔧 CONFIGURACIÓN PARA CASOS COMUNES
════════════════════════════════════════════════════════════════════════════

CASO 1: Mark un día como ocupada por reserva
─────────────────────────────────────────

En el evento onSubmit del formulario, agrega:

const fechaReserva = document.getElementById('fecha').value;
SISTEMA_DISPONIBILIDAD.agregarDiaOcupado(fechaReserva);

CASO 2: Marcar fin de semana como cerrada
──────────────────────────────────────────

Modifica getHorariosParaDia para devolver array vacío los domingos:

getHorariosParaDia(fecha) {
    const diaSemana = fecha.getDay();
    
    // Domingos: cerrada
    if (diaSemana === 0) return [];
    
    // Sábados: fin de semana normal
    if (diaSemana === 6) return this.horariosDisponibles.finDeSemana;
    
    // Lunes-Viernes: normal
    return this.horariosDisponibles.normal;
}

CASO 3: Sincronizar con backend
────────────────────────────────

Agregado a inicialización:

async function inicializarDisponibilidad() {
    try {
        const datos = await fetch('/api/disponibilidad').then(r => r.json());
        SISTEMA_DISPONIBILIDAD.diasOcupados = datos.diasOcupados;
    } catch (error) {
        console.warn('Backend no disponible, usando local');
    }
}
