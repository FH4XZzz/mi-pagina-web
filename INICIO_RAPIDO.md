# 🚀 INICIO RÁPIDO - OG BEAT PRODUCTION

## ⚡ En 3 Minutos

### 1️⃣ Abrir la Página
- Abre el archivo `index.html` en tu navegador web
- ¡La página cargará completamente en tu navegador!

### 2️⃣ Personalizar la Información
Edita los siguientes archivos con un editor de texto (Notepad++, VS Code, Sublime):

**En `index.html`:**
- Línea ~393: Cambiar número WhatsApp
- Línea ~431: Cambiar email de contacto
- Línea ~425: Cambiar número de teléfono
- Línea ~421: Cambiar ubicación

**Ejemplo:**
```html
Busca:  <a href="https://wa.me/1234567890"
Cambia a: <a href="https://wa.me/+541234567890"  (tu número real)
```

### 3️⃣ Probar el Formulario
1. Abre `index.html` en navegador
2. Desplázate a la sección "Reserva Tu Cita"
3. Prueba llenar y enviar el formulario
4. Verifica que aparezca el mensaje de confirmación

### 4️⃣ Desplegar Online
Elige una opción:

**Opción A - Netlify (Más fácil)**
1. Ve a netlify.com
2. Arrastra la carpeta `og-beat-production`
3. ¡Listo! Tu página está online

**Opción B - GitHub Pages**
1. Sube archivos a GitHub
2. Habilita GitHub Pages
3. ¡Listo!

## 🌐 ¿Cómo Desplegar?

### Opción 1: Netlify (Recomendado)
```
1. Ve a netlify.com
2. Sign up (gratis)
3. "Add new site" → "Deploy manually"
4. Arrastra la carpeta aquí
5. ¡Listo! Te dará un link
```

### Opción 2: Vercel
```
1. Ve a vercel.com
2. Sign up (gratis)
3. "New Project"
4. Conecta repositorio GitHub
5. Deploy automático
```

### Opción 3: Tu Propio Hosting
```
1. Compra hosting (Hostinger, SiteGround, etc.)
2. Descarga archivos por FTP
3. Sube archivos a htdocs/
4. Accede a tu dominio
```

## 📞 Soporte

### Si algo sale mal:

1. **Abre la consola** (F12 → Console)
2. **Lee los errores**
3. **Compara con index.html original**
4. **Limpia caché** (Ctrl + Shift + Del)

### Errores comunes:

| Error | Solución |
|-------|----------|
| No carga nada | Verifica que abras `index.html` |
| Botones no funcionan | Verifica JavaScript está habilitado |
| Se ve feo | Abre en pantalla completa (F11) |
| Iconos no aparecen | Requiere conexión a internet |
