ERROR: "RangeError: Invalid time value"
UBICACIÓN: script.js línea 329
CAUSA: Intento de crear Date object con formato incorrecto

DESCRIPCIÓN DETALLADA:
─────────────────────────────────────────────────────────────────────────────

Flatpickr devuelve fechas en formato: d/m/Y
Ejemplo: "17/03/2026"

El código intentaba hacer:
  const fecha = new Date("17/03/2026" + "T00:00:00");
  // Resultado: new Date("17/03/2026T00:00:00") ❌ INVÁLIDO

JavaScript interpreta dates en formato ISO como: YYYY-MM-DD
Ejemplo válido: "2026-03-17"

Correcto:
  const fecha = new Date("2026-03-17T00:00:00"); ✅ VÁLIDO
