🧪 PRUEBAS BÁSICAS DE FUNCIONALIDAD
════════════════════════════════════════════════════════════════════════

[ ] 1. ABRIR LA PÁGINA
   • Abre index.html en tu navegador preferido
   • Espera a que cargue completamente
   • Desplázate a la sección "Reserva Tu Cita"

[ ] 2. VERIFICAR APARIENCIA DEL CAMPO
   • El campo de fecha debe tener placeholder: "Haz clic para seleccionar una fecha"
   • El campo debe tener el mismo estilo que los demás inputs
   • Debe decir debajo: "Disponible desde mañana en adelante"

[ ] 3. ABRIR EL CALENDARIO
   • Haz clic en el campo de fecha
   • El calendario debe abrirse con una animación suave
   • Debe aparecer ABAJO o ARRIBA del campo (depende del espacio disponible)

[ ] 4. VERIFICAR TEMA OSCURO
   • El calendario debe tener fondo oscuro (#0a0e27, #192033, #1f273f)
   • Los colores deben combinar con el sitio
   • Las transiciones deben ser suaves

[ ] 5. VERIFICAR DÍAS BLOQUEADOS
   ✓ HOJA DE HOY debe estar DESHABILITADA
     • Debe aparecer con baja opacidad/palida
     • No debe poder seleccionarse
     • Al hacer hover, NO debe cambiar
   
   ✓ FECHAS PASADAS deben estar DESHABILITADAS
     • Si es 17 de marzo, todos los días anteriores no deben ser seleccionables
     • Deben aparecer apagados/grises
   
   ✓ MAÑANA EN ADELANTE debe estar DISPONIBLE
     • Si es 17 de marzo, desde el 18 en adelante debe poder seleccionarse
     • Al hacer hover sobre un día disponible, debe cambiar el color

[ ] 6. SELECCIONAR UNA FECHA
   • Haz clic en una fecha (ej: 20 de marzo)
   • La fecha debe aparecer en el campo: "20/03/2026"
   • El calendario debe cerrarse automáticamente

[ ] 7. VERIFICAR DÍA ACTUAL (HOY)
   • Si hoy es 17 de marzo, el día 17 debe:
     • Tener UN BORDE VERDE NEÓN (#00FF88)
     • Estar escrito en verde neón
     • Ser muy visible pero NO seleccionable

[ ] 8. VERIFICAR DÍA SELECCIONADO
   • El día que seleccionaste debe:
     • Tener fondo DORADO (#FFD700)
     • Estar en color NEGRO
     • Tener sombra dorada
     • Ser muy visible

[ ] 9. CAMBIAR DE MES
   • Haz clic en el campo de fecha para abrir el calendario
   • Busca botones para navegar: < (anterior) y > (siguiente)
   • Haz clic para cambiar de mes
   • El mes/año debe actualizar correctamente
   • Las animaciones deben ser suaves

[ ] 10. VERIFICAR VALIDACIÓN
   • Intenta enviar el formulario sin seleccionar fecha
   • Debe mostrar un mensaje de error: "Por favor selecciona una fecha"
   • El campo debe tener borde rojo

🆘 SOLUCIÓN DE PROBLEMAS
════════════════════════════════════════════════════════════════════════

SI EL CALENDARIO NO APARECE:
   1. Abre la consola del navegador (F12)
   2. Busca errores en la pestaña "Console"
   3. Si dice "flatpickr is not defined", el CDN no cargó
       → Verifica conexión a internet
       → Recarga la página con Ctrl+Shift+R (limpiar caché)

SI LOS COLORES NO VEN BIEN:
   1. Verifica que styles.css cargó correctamente
   2. Abre DevTools y busca en la pestaña "Styles"
   3. Busca "flatpickr" en los estilos
   4. Si no aparece, recarga la página

SI NO SE BLOQUEAN LAS FECHAS:
   1. Abre la consola (F12)
   2. Escribe: new Date().toISOString().split('T')[0]
   3. Debe mostrarte hoy's fecha
   4. Si aparece error, hay problema con JavaScript

SI EL CALENDARIO SÍ APARECE PERO SE VE FENO:
   1. Limpia caché del navegador (Ctrl+Shift+Del)
   2. Cierra el navegador completamente
   3. Abre nuevamente index.html
