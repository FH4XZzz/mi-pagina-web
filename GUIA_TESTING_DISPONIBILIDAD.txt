🎯 OBJETIVO
════════════════════════════════════════════════════════════════════════════

Esta guía te permite probar el sistema de disponibilidad sin necesidad de 
esperar a que se cumpla una fecha. Usa estos ejemplos en la consola.

📝 PASOS RÁPIDOS DE TESTING
════════════════════════════════════════════════════════════════════════════

1. ABRE EL SITIO
   └─ Abre index.html en tu navegador
   
2. ABRE LA CONSOLA
   └─ Presiona: F12 (Windows/Linux) o Cmd+Option+I (Mac)
   └─ Ve a la pestaña "Console"
   
3. COPIA Y PEGA LOS EJEMPLOS
   └─ Usa los ejemplos de abajo
   └─ Presiona Enter
   
4. VE A LA SECCIÓN "RESERVA TU CITA"
   └─ Desplázate o haz clic en el link
   └─ Haz clic en el campo de fecha
   └─ Verifica los cambios en el calendario

🔴 EJEMPLO 1: Marcar varios días como ocupados
════════════════════════════════════════════════════════════════════════════

Copia y pega esto en la consola:

```javascript
// Obtener hoy y las próximas fechas
const hoy = new Date();

// Marcar los próximos 3 días como ocupados
SISTEMA_DISPONIBILIDAD.diasOcupados = [
    new Date(hoy.getTime() + 2 * 24 * 60 * 60 * 1000).toISOString().split('T')[0],
    new Date(hoy.getTime() + 4 * 24 * 60 * 60 * 1000).toISOString().split('T')[0],
    new Date(hoy.getTime() + 6 * 24 * 60 * 60 * 1000).toISOString().split('T')[0]
];

console.log('📅 Días ocupados:', SISTEMA_DISPONIBILIDAD.diasOcupados);
```

✅ RESULTADO ESPERADO:
   - En la consola verás las 3 fechas ocupadas
   - En el calendario aparecerán en ROJO esos días
   - No podrás seleccionar esos días

🟢 EJEMPLO 2: Limpiar días ocupados
════════════════════════════════════════════════════════════════════════════

Copia y pega esto en la consola:

```javascript
// Limpiar todos los días ocupados
SISTEMA_DISPONIBILIDAD.diasOcupados = [];

console.log('✅ Todos los días están disponibles ahora');
```

✅ RESULTADO ESPERADO:
   - Los días rojos desaparecerán
   - Todos los días desde mañana estarán verdes
   - Podrás seleccionar cualquier día futuro

⏰ EJEMPLO 3: Verificar horarios para diferentes días
════════════════════════════════════════════════════════════════════════════

Copia y pega esto en la consola:

```javascript
// Horarios para un lunes (día 0 = domingo, 1 = lunes)
const lunes = new Date();
lunes.setDate(lunes.getDate() + ((1 - lunes.getDay() + 7) % 7 || 7));

// Horarios para un sábado (día 6 = sábado)
const sabado = new Date();
sabado.setDate(sabado.getDate() + ((6 - sabado.getDay() + 7) % 7 || 7));

console.log('📅 Lunes:', lunes.toLocaleDateString());
console.log('🕐 Horarios:', SISTEMA_DISPONIBILIDAD.getHorariosParaDia(lunes));
console.log('');
console.log('📅 Sábado:', sabado.toLocaleDateString());
console.log('🕐 Horarios:', SISTEMA_DISPONIBILIDAD.getHorariosParaDia(sabado));
```

✅ RESULTADO ESPERADO:
   - Verás los 2 arrays de horarios diferentes
   - Lunes: Empieza a 8:00 AM (14 opciones)
   - Sábado: Empieza a 10:00 AM (12 opciones)

🎨 EJEMPLO 4: Ver el estado actual del sistema
════════════════════════════════════════════════════════════════════════════

Copia y pega esto en la consola:

```javascript
// Mostrar estado completo del sistema
console.table({
    'Días ocupados': SISTEMA_DISPONIBILIDAD.diasOcupados.length || 'Ninguno',
    'Ocupados (lista)': SISTEMA_DISPONIBILIDAD.diasOcupados.join(', '),
    'Horarios normales': SISTEMA_DISPONIBILIDAD.horariosDisponibles.normal.length + ' horas',
    'Horarios fin de semana': SISTEMA_DISPONIBILIDAD.horariosDisponibles.finDeSemana.length + ' horas'
});
```

✅ RESULTADO ESPERADO:
   - Tabla con el estado actual del sistema
   - Verás cuántos días están ocupados
   - Verás cuántos horarios hay por día

+️➕ EJEMPLO 5: Agregar/Quitar un día ocupado específico
════════════════════════════════════════════════════════════════════════════

AGREGAR UN DÍA COMO OCUPADO:

```javascript
// Marcar mañana como ocupado
const hoy = new Date();
const mañana = new Date(hoy.getTime() + 24 * 60 * 60 * 1000);
const mañanaISO = mañana.toISOString().split('T')[0];

SISTEMA_DISPONIBILIDAD.agregarDiaOcupado(mañanaISO);
console.log('✅ Mañana marcado como ocupado');
```

QUITAR UN DÍA DE OCUPADA:

```javascript
// Desmarcar mañana
const hoy = new Date();
const mañana = new Date(hoy.getTime() + 24 * 60 * 60 * 1000);
const mañanaISO = mañana.toISOString().split('T')[0];

SISTEMA_DISPONIBILIDAD.quitarDiaOcupado(mañanaISO);
console.log('✅ Mañana marcado como disponible');
```

🔍 EJEMPLO 6: Verificar si un día está ocupada
════════════════════════════════════════════════════════════════════════════

```javascript
// Verificar si 20 de marzo está ocupado
const estaOcupado = SISTEMA_DISPONIBILIDAD.getDiaOcupado('2026-03-20');
console.log('¿2026-03-20 está ocupado?', estaOcupado);

// Agregar y verificar
SISTEMA_DISPONIBILIDAD.agregarDiaOcupado('2026-03-20');
console.log('¿2026-03-20 está ocupado ahora?', SISTEMA_DISPONIBILIDAD.getDiaOcupado('2026-03-20'));
```

✅ RESULTADO ESPERADO:
   - Primera verificación: false
   - Después de agregar: true

📊 EJEMPLO 7: Scenario completo de reserva
════════════════════════════════════════════════════════════════════════════

Simula un flujo completo de reserva:

```javascript
// 1. Ver disponibilidad inicial
console.log('=== INICIO ===');
console.log('Días ocupados:', SISTEMA_DISPONIBILIDAD.diasOcupados.length);

// 2. Alguien reserva mañana
const hoy = new Date();
const fechaReserva = new Date(hoy.getTime() + 24 * 60 * 60 * 1000)
    .toISOString()
    .split('T')[0];

SISTEMA_DISPONIBILIDAD.agregarDiaOcupado(fechaReserva);
console.log('✅ Cliente reservó:', fechaReserva);

// 3. Ver disponibilidad después
console.log('Días ocupados ahora:', SISTEMA_DISPONIBILIDAD.diasOcupados.length);
console.log('Fechas ocupadas:', SISTEMA_DISPONIBILIDAD.diasOcupados.join(', '));

// 4. Cliente cancela
SISTEMA_DISPONIBILIDAD.quitarDiaOcupado(fechaReserva);
console.log('❌ Cliente canceló');
console.log('Días ocupados finales:', SISTEMA_DISPONIBILIDAD.diasOcupados.length);
```

🎬 VERIFICACIÓN VISUAL FINAL
════════════════════════════════════════════════════════════════════════════

Después de ejecutar cualquier ejemplo:

1. Desplázate a "Reserva Tu Cita"
2. Haz clic en el campo de fecha
3. Verifica visualmente:
   ✓ Días verdes = disponibles (clicables)
   ✓ Días rojos = ocupados (no clicables)
   ✓ Punto verde en esquina = disponible
   ✓ Punto rojo en esquina = ocupado
   ✓ Leyenda debajo del campo

🔁 CICLO DE TESTING RECOMENDADO
════════════════════════════════════════════════════════════════════════════

1. INICIO
   └─ Limpia días ocupados (Ejemplo 2)
   
2. MARCA DÍAS OCUPADOS
   └─ Usa Ejemplo 1
   └─ Abre el calendario
   └─ Verifica que aparecen en rojo
   
3. TRANSICIÓN A DIFERENTES FECHAS
   └─ Selecciona días verdes
   └─ Observa que los horarios cambian (o no, según config)
   
4. FINAL
   └─ Limpia todo (Ejemplo 2)
   └─ Restaura el estado inicial

💡 TIPS DE DEBUGGING
════════════════════════════════════════════════════════════════════════════

Si algo no funciona:

1. ¿El objeto existe?
   console.log(SISTEMA_DISPONIBILIDAD);

2. ¿Las fechas están bien?
   console.log(new Date().toISOString().split('T')[0]);

3. ¿Flatpickr está cargado?
   console.log(window.flatpickr);

4. ¿El calendario está inicializado?
   console.log(document.querySelector('.flatpickr-calendar'));

5. Refresca el navegador (Ctrl+Shift+R) para limpiar caché
